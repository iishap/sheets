# Google Sheets Clone

A web-based application built to replicate the functionality of **Google Sheets** using **TypeScript**, **JavaScript**, and **CSS**.

##  Features
- Create and Edit Spreadsheets
- Multiple Sheets Support
- Cell Formulas (Basic Arithmetic Operations)
- Row and Column Resizing
- Merge Cells
- Real-Time Data Validation
- Export Sheets as CSV
- Undo/Redo Functionality

##  Technologies Used
- **TypeScript**
- **JavaScript**
- **CSS**

##  Installation

### Prerequisites
- Node.js installed
- Git installed

### Clone the Repository
```bash
git clone <repo-url>
cd google-sheets-clone
```

### Install Dependencies
```bash
npm install
```

### Run the Application
```bash
npm start
```

##  How to Use
1. Enter data into cells by clicking and typing.
2. Use formulas like `=SUM(A1:A10)` to calculate values.
3. Resize rows and columns by dragging borders.
4. Export your sheet using the **Download CSV** button.
